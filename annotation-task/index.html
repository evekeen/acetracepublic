Automatic Tracking

The tracking allows to compute the object trajectory automatically. We already have a working prototype, however to improve our algorithm we need to prepare more data for the machine learning system.

The job

You will need to annotate a set of disc golf videos. In every video a disc location has to be marked by a rectangle. Annotation process is done via CVAT tool which will be installed to your computer. It’s 300-400 videos total. For 100 videos annotated you’ll get a lifetime Ace Trace Premium subscription plus a little cash bonus. If you do all 400 videos, we will offer you a bonus of 150$.

Example of the annotation process of one video: https://drive.google.com/file/d/1KAfYsa7sgoJmjTK-fVSJaavpJPh2YQzJ/view?usp=sharing
